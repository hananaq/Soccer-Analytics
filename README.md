# Soccer Analytics Dashboard

Soccer Pass Decision Analytics Dashboard — an interactive visualization tool that presents the results of advanced pass analysis in soccer.

This repository focuses on displaying insights and results, and does not include the underlying analysis pipeline or models.

---

## Project Overview

The main objective of this project is to:

- Analyze unsuccessful passes  
- Understand why passes fail  
- Provide interactive visual insights into decision-making during matches  

---

## Dataset

The dataset used for this analysis was obtained from SkillCorner.
